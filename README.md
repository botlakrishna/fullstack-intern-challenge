# FullStack Intern Coding Challenge

## Tech Stack
- Backend: ExpressJS
- Database: PostgreSQL
- Frontend: ReactJS

## Features
- Admin, Store Owner, Normal User roles
- Login/Register
- Rating stores
- Dashboards for all users
- Update password

## How to Run
1. `cd backend` → `npm install` → `npm start`
2. `cd frontend` → `npm install` → `npm start`
